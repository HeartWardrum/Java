## 问题：相同类名引入，如何使用？

### 答：遵循以下原则：

前置准备：

~~~~java
package com.a;
public class Test(){
    public int num1 = 1;
}
~~~~

~~~~java
package com.b;
public class Test(){
public int num2 = 2;
}
~~~~



1. 使用声明更具体的包的类

~~~~java
import com.a.*;
import com.b.Test:
//使用b中的Test类
~~~~

2. 导入包声明程度一样具体——只能使用第一个，第二个直接报错

~~~~java
import com.a.Test;
import com.b.Test;
//只能使用a中的Test；
~~~~

3. 不同导入包都采用通配符声明——必须进行选择才可正常使用

~~~~java
import com.a.*;
import com.b.*;
public static void main(String[] args){
    com.a.Test a = new com.a.Test();
    Test b = new Test();
    //只能用a中的Test
}
~~~~
